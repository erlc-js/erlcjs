## Why Choose ERLCjs?

<CardGrid stagger>
    <Card title="TypeScript Native" icon="seti:typescript">
        Built from the ground up in TS, providing robust types, autocompletion, and compile-time checks for all raw payloads.
    </Card>
    <Card title="Automatic Rate Limiting" icon="rocket">
        Queues API requests automatically and respects ER:LC's rate limit reset headers to prevent 429 errors.
    </Card>
    <Card title="Real-time Event System" icon="forward">
        Subscribe to server updates, player joins/leaves, vehicle spawns, mod calls, and kill logs using polling or webhooks.
    </Card>
    <Card title="Developer-Friendly API" icon="code">
        Interact with players and vehicles using intuitive object-oriented methods like `player.kick()` and `player.message()`.
    </Card>
</CardGrid>
