title: Managing Players

Tracking and managing players is one of erlc.js's core strengths. When polling or gateway mode is enabled, the client maintains a real-time player cache under `client.players.cache`.

## Accessing Players

Active players are stored inside a Map keyed by their Roblox UserId:

```javascript
// Access the players cache
const playerCache = client.players.cache;

// Get a specific player by UserId
const player = playerCache.get(1234567);

if (player) {
    console.log(`Player Name: ${player.username}`);
    console.log(`Current Team: ${player.team}`);
    console.log(`Permission Level: ${player.permission}`);
}
```

### Player Properties

Each `Player` structure exposes several properties parsed from the raw API:
*   `id`: The Roblox UserId.
*   `username`: The Roblox username.
*   `team`: The current team in-game (e.g. Civilians, Police, Sheriff, Fire).
*   `permission`: Server permissions (`Normal`, `Server Moderator`, `Server Administrator`, `Server Owner`).
*   `wantedLevel`: The wanted star rating (0 to 5).
*   `callsign`: The player's department callsign if active.
*   `location`: Spatial location details containing `x`, `y`, `postalCode`, `streetName`, and `buildingNumber`.

## Modifying Player States (Moderator Actions)

The `Player` class exposes shortcut methods that compile and execute game commands. These actions require appropriate admin credentials.

### Kick a Player

Kick a player from the server with an optional custom reason:

```javascript
client.on(ERLCEvents.playerJoin, async (player) => {
    if (player.username === 'SpammerUser') {
        await player.kick('Automated Ban: Flagged Username');
        console.log(`Kicked user ${player.username}`);
    }
});
```

### Kill a Player

Instantly kill the player in-game (resets their character):

```javascript
const player = client.players.cache.get(userId);
if (player) {
    await player.kill();
}
```

### Message a Player (PM)

Sends an in-game private message to the player's screen:

```javascript
const player = client.players.cache.get(userId);
if (player) {
    await player.message('Welcome back! Remember to follow the speed limits.');
}
```
