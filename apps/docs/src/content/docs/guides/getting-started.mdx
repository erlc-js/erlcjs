Welcome to the **ERLCjs** getting started guide! ERLCjs is a developer-friendly library designed to help you interact with the Roblox Emergency Response: Liberty County private server API.

This guide will walk you through setting up your project, obtaining API keys, installing the library, and running your first server script.

## Quick Setup

Follow these steps to get your first script running in minutes.

<Steps>

1.  ### Obtain Your ER:LC Server Key

    To use ERLCjs, you must have an API Server Key generated from your ER:LC private server settings.
    *   Open Roblox and join your ER:LC Private Server.
    *   Open the server settings panel (requires Owner or co-owner permissions).
    *   Navigate to **Server API** or developer settings.
    *   Generate a new API key and copy it somewhere safe. Keep this key secret!

2.  ### Initialize Your Node.js Project

    Create a new directory and initialize a Node.js project:
    ```bash
    mkdir erlc-bot
    cd erlc-bot
    npm init -y
    ```
    It is recommend to configure your `package.json` to use ES modules by adding `"type": "module"`.

3.  ### Install ERLCjs

    Install the core package using your preferred package manager:
    <Tabs>
      <TabItem label="npm">
        ```bash
        npm install @erlcjs/core
        ```
      </TabItem>
      <TabItem label="pnpm">
        ```bash
        pnpm add @erlcjs/core
        ```
      </TabItem>
      <TabItem label="yarn">
        ```bash
        yarn add @erlcjs/core
        ```
      </TabItem>
    </Tabs>

4.  ### Write Your First Script

    Create a file named `index.js` and paste the following code:
    <Tabs>
        <TabItem label="esm">
            ```javascript
            import { Client, ERLCEvents } from '@erlcjs/core';

            // Initialize the ERLCjs Client
            const client = new Client({
                serverKey: 'YOUR_SERVER_API_KEY', // Replace with your copied key
                polling: true // Enable periodic polling
            });

            // Listen for the player join event
            client.on(ERLCEvents.playerJoin, (player) => {
                console.log(`[JOIN] ${player.username} has entered the server!`);
                player.message('Welcome to our private server! Have fun.');
            });

            // Handle connection/API errors
            client.on('error', (err) => {
                console.error('An API error occurred:', err);
            });

            console.log('ERLCjs bot is running...');
            ```
        </TabItem>
        <TabItem label="cjs">
            ```javascript
            const { Client, ERLCEvents } = require('@erlcjs/core');

            // Initialize the ERLCjs Client
            const client = new Client({
                serverKey: 'YOUR_SERVER_API_KEY', // Replace with your copied key
                polling: true // Enable periodic polling
            });

            // Listen for the player join event
            client.on(ERLCEvents.playerJoin, (player) => {
                console.log(`[JOIN] ${player.username} has entered the server!`);
                player.message('Welcome to our private server! Have fun.');
            });

            // Handle connection/API errors
            client.on('error', (err) => {
                console.error('An API error occurred:', err);
            });

            console.log('ERLCjs bot is running...');
            ```
        </TabItem>
    </Tabs>
    

5.  ### Run the Script

    Run the script using Node.js:
    ```bash
    node index.js
    ```

</Steps>
